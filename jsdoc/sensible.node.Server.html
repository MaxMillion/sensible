onHTTPRequest(inRequest, outResponse, inRequestURL, inRequestParams)

Called on receipt of an HTTP request. Calls the REST dispatcher to try and resolve the request against our delegate. If the resolution succeeds, the handler is called and the response assumed to be JSON. If the resolution fails, we attempt to serve a file with the appropriate path.
